========================================
  Vercel 배포 가이드 (API 키 숨김 버전)
========================================

이 버전은 API 키가 서버에서만 사용되어 수강생에게 노출되지 않습니다.

--------------------
1단계: Vercel 가입
--------------------
1. https://vercel.com 접속
2. "Sign Up" 클릭
3. GitHub 계정으로 가입 (가장 쉬움)

--------------------
2단계: GitHub에 코드 업로드
--------------------
1. GitHub에서 새 저장소(Repository) 생성
2. 이 폴더(vercel-version) 안의 모든 파일 업로드:
   - api/recommend.js
   - public/index.html
   - vercel.json
   - package.json

--------------------
3단계: Vercel에서 프로젝트 생성
--------------------
1. Vercel 대시보드에서 "Add New Project" 클릭
2. GitHub 저장소 선택
3. "Import" 클릭

--------------------
4단계: 환경변수 설정 (중요!)
--------------------
1. 프로젝트 설정(Settings) → Environment Variables
2. 새 변수 추가:
   - Name: CLAUDE_API_KEY
   - Value: sk-ant-api... (본인의 API 키)
3. "Save" 클릭

--------------------
5단계: 배포
--------------------
1. "Deploy" 버튼 클릭
2. 배포 완료되면 URL 받음 (예: https://your-project.vercel.app)
3. 이 URL을 수강생에게 공유!

========================================
  완료!
========================================

수강생은 URL만 접속하면 바로 사용 가능
API 키는 Vercel 서버에만 저장되어 안전함

문제 발생 시:
- Vercel 대시보드 → Logs에서 오류 확인
- Environment Variables가 제대로 설정되었는지 확인
